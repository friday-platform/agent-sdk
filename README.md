# Friday Agent SDK

Write AI agents in Python (and soon other languages) that run inside the Friday platform. Agents call LLMs, make HTTP requests, and use MCP tools through the host—no API keys or dependencies required.

**Full documentation:** https://docs.hellofriday.ai/

## Quick start

See [`packages/python/README.md`](packages/python/README.md) to install the Python SDK and run your first agent.
