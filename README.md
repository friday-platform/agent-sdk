# Friday Agent SDK

Write AI agents in Python (and soon other languages) that run inside Friday's WebAssembly sandbox. Agents call LLMs, make HTTP requests, and use MCP tools through the host—no API keys or dependencies required.

**Full documentation:** https://platform.hellofriday.ai/docs/sdk/overview

## Quick start

See [`packages/python/README.md`](packages/python/README.md) to install the Python SDK and run your first agent.
